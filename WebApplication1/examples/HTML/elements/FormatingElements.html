Text Stringth

Bold element (<b>) can be used to give more importance to text, but strong element (<strong>) tells the reading tools that the text is really strong. The opposite of strong element is the small element (<small>) which gives less importance to text.

Text emphasis

The <em> element represents stress emphasis of its contents, while the <i> element represents text that is set off from the normal prose.

Deleted and inserted text

The text that is no longer correct should look stroked using the <s> element, but the <del> element gives additional meaning that the text has been deleted from a document. The opposite of that is the pair <u> element for underlined text and <ins> element that defines a text that has been inserted into a document.
